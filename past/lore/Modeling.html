--UML--
モデリングといえば、UML(Unified Modeling Language) を知っておく必要があります。UML ( Unified Modeling Language ) は OMG ( Object Management Group ) (OMG Japan)という標準化団体で規定されているソフトウエア開発におけるモデ リング言語です。
  • ユースケース図（use case diagram）
  • クラス図（class diagram）
  • behavior diagrams:
    • 状態チャート図（statechart diagram）
    • アクティビティ図（activity diagram）
    • interaction diagrams:
      • シーケンス図（sequence diagram）
      • コラボレーション図（collaboration diagram）
  • 実装モデル（implementation diagrams）:
    • コンポーネント図（component diagram）
    • 展開図（deployment diagram）
