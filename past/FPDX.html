PDX着信メロディ送信アプリケーション

Javaページへ

PXD.jar

使うには上のPDX.jarをダウンロードし、下のコマンドをDOSプロンプトで実行すると上のようなウィンドウがでてきます。

#> java -cp PDX.jar FPDX

そして"SERVER"にはメールサーバ名を(例：mail.dion.ne.jp)、"TO", "FROM"にはメールアドレスを、"MELODY"にはalsファイルを入力し、SENDボタンを押すとメロディが送信されます。

alsファイルとはH"のメロディファイルで、H"でメロディを添付してパソコンにメールを送るとテキストファイルがついてきます。そのテキスト部分だけを切り取ったテキストファイルをalsファイルと呼んでいます。

alsファイル

ちなみに以下のようなコマンドでもメロディを送信することができます。

#> java -cp PDX.jar PDX
